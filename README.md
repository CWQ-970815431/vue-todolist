# 	Vue-todolist

[Vue官方文档](https://cn.vuejs.org/v2/guide/) 了解一下

1. 前置项
   安装Node

2. 然后全局安装 [Vue CLI](https://cli.vuejs.org/zh/guide/installation.html)
   

   ```
   npm install -g @vue/cli
   ```

3. 创建项目

   ```
   vue create vue-todolist
   ```

4. 启动CLI服务

   ```
   cd vue-todolist
   $ npm run serve
   ```
